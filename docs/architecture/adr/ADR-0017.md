# ADR-0017 — Policies como código derivadas do grafo

Status: Accepted  
Data: 2026-02-10

## Contexto
Policies precisam ser auditáveis, determinísticas e explicáveis.

## Decisão
- Policies são funções puras sobre o grafo (IR)
- Sem efeitos colaterais
- Produzem findings com evidência (entidades/caminhos)
- Versionadas e testáveis

## Consequências
- Governança forte
- Evidência clara
- Menos regressão e ambiguidade
