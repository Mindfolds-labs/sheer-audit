# ADR-0005 — Workspace e I/O restritos (sandbox de execução)

Status: Accepted  
Data: 2026-02-10

## Contexto
Ferramentas de análise não devem vazar dados nem escrever fora do permitido.

## Decisão
Toda execução usa um workspace controlado:
- leitura: somente repositório alvo (somente leitura)
- escrita: somente diretório de artefatos (outputs)
- proíbe escrita fora do workspace
- proíbe acesso a secrets por padrão

## Consequências
- Redução de risco operacional
- Menos incidentes por “efeito colateral”
- Melhor aderência a compliance
