# ADR-0001 — Sheer: objetivo e escopo do v1

Status: Accepted  
Data: 2026-02-10

## Contexto
Precisamos de valor imediato (v1) sem retrabalho estrutural para v2+.

## Decisão
O v1 entrega contratos e fluxo base:
- ingestão de repositório (somente leitura)
- construção do IR (grafo semântico)
- execução de policies determinísticas
- geração de relatórios derivados (console/JSON/MD/HTML/CSV/SARIF)

O v1 NÃO entrega:
- execução remota (sheer-agent) como padrão
- auditoria central obrigatória (audit server)

## Consequências
- Entrega rápida e útil
- Contratos estáveis para evolução (v2+)
- Base defensável para segurança
