# ADR-0014 — Topologia futura: CLI, Agent e Audit Server

Status: Accepted  
Data: 2026-02-10

## Contexto
Algumas organizações exigem auditoria central e execução restritiva.

## Decisão
Arquitetura alvo:
- sheer — CLI
- sheer-agent — execução restritiva
- sheer-audit-server — ingestão, policies e trilha auditável

O v1 entrega contratos; v2+ adiciona centralização.

## Consequências
- Valor imediato no v1
- Evolução segura no v2+
- Nenhum retrabalho estrutural
