# ADR-0007 — Execução limitada e segura (restrictive por padrão em contextos remotos)

Status: Accepted  
Data: 2026-02-10

## Contexto
Execução em ambientes remotos aumenta risco de abuso e quebra de segurança.

## Decisão
Em contexto remoto (v2+):
- usar profile restrictive como padrão
- permitir somente pipelines autorizados
- bloquear comandos arbitrários
- bloquear “shell remoto”
- bloquear alteração de policy sem assinatura

## Consequências
- Segurança por design
- Menos surpresas em produção
- Base direta para agent restritivo
