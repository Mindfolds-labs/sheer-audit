# ADR-0013 — Relatórios como artefatos derivados

Status: Accepted  
Data: 2026-02-10

## Contexto
UI não pode ser o core.

## Decisão
Relatórios e dashboards são derivados do IR:
- console
- JSON
- Markdown
- HTML
- CSV
- SARIF

## Consequências
- Evolução livre de formatos
- Integração fácil com ferramentas externas
