# ADR-0008 — Agent restritivo não toma decisões

Status: Accepted  
Data: 2026-02-10

## Contexto
Agentes remotos genéricos são riscos de segurança.

## Decisão
O agent (v2+):
- executa pipeline autorizado
- coleta resultados
- envia para auditoria central

É proibido:
- shell remoto
- comandos arbitrários
- alteração de policy sem assinatura

## Consequências
- Superfície de ataque mínima
- Aprovação facilitada por times de segurança
- Arquitetura defensável
