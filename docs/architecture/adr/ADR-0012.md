# ADR-0012 — Avaliação de caminhos proibidos (reachability)

Status: Accepted  
Data: 2026-02-10

## Contexto
Violações reais ocorrem por caminhos indiretos.

## Decisão
Avaliar reachability via BFS/DFS para detectar:
- caminhos proibidos
- dependências indiretas

## Consequências
- Governança mais forte
- Evidência clara do problema
