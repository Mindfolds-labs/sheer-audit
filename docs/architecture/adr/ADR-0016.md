# ADR-0016 — IA / LLM como consumidor do IR (não executor)

Status: Accepted  
Data: 2026-02-10

## Contexto
IA pode agregar valor, mas não pode aumentar risco operacional.

## Decisão
IA/LLM:
- somente leitura do IR e artefatos derivados
- pode explicar, resumir, sugerir hipóteses (não decisões)

É proibido:
- executar código analisado
- executar shell
- alterar policies
- gerar findings oficiais
- escrever de volta no IR

## Consequências
- Zero aumento de superfície de ataque
- Copiloto seguro para humanos
- Compliance preservado
