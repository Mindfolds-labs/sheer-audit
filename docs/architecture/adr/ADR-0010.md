# ADR-0010 — Coleta de métricas passiva

Status: Accepted  
Data: 2026-02-10

## Contexto
Ferramentas de auditoria não podem degradar ambientes.

## Decisão
Métricas são:
- passivas
- não invasivas
- sem execução de código analisado

## Consequências
- Baixo risco operacional
- Base segura para evolução futura
