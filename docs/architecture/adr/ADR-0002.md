# ADR-0002 — Perfis de execução fazem parte do manifesto

Status: Accepted  
Data: 2026-02-10

## Contexto
Ambientes corporativos exigem restrições claras e previsíveis.

## Decisão
O sistema adota perfis explícitos e imutáveis por execução (ex.: restrictive).
O perfil:
- define o conjunto de pipelines permitidos
- define limites de I/O (workspace)
- define formatos/saídas autorizadas
- é enviado junto com resultados para auditoria

## Consequências
- Comportamento previsível
- Menos surpresas em produção
- Base direta para agent futuro
