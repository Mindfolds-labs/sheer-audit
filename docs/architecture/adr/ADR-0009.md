# ADR-0009 — Identidade mínima por máquina

Status: Accepted  
Data: 2026-02-10

## Contexto
Autenticação precisa ser simples, revogável e auditável.

## Decisão
Identidade baseada em:
- machine_id
- profile
- tool/agent version
- token rotativo

## Consequências
- Operação simples no início
- Evolução futura para IAM sem quebra
