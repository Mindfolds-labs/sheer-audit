# ADR-0004 — CI: gates mínimos obrigatórios

Status: Accepted  
Data: 2026-02-10

## Contexto
Sem gates mínimos, a qualidade degrada e quebra em produção aumenta.

## Decisão
O repositório deve ter gates mínimos em CI:
- lint/format (automático)
- testes (unit + smoke/integration do pipeline)
- verificação de build/empacotamento (quando aplicável)
- (opcional) typecheck, conforme stack

Falhas em gate bloqueiam merge no main.

## Consequências
- Menos regressões
- Maior previsibilidade
- Padronização e disciplina de qualidade
