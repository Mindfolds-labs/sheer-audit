# ADR-0006 — Modelo de dados: evidência primeiro

Status: Accepted  
Data: 2026-02-10

## Contexto
Findings precisam ser explicáveis, auditáveis e baseados em evidência.

## Decisão
Todo finding deve incluir:
- evidência (origem no IR + caminho/entidade)
- regra/policy que gerou
- severidade e recomendação
- referência versionada (IR/policy/tool)

## Consequências
- Auditoria objetiva
- Explicabilidade consistente
- Menos discussão subjetiva sobre achados
