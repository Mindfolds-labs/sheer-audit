SHEER AUDIT - COMANDOS

INSTALAÇÃO:
  pip install -e .

AUDITORIA AVANÇADA:
  sheer advanced --full-scan --ieee --export docs/sheer_audit
  sheer audit-secure --repo-path . --vault-path docs/sheer_audit/vault/audit.sheerdb

SNAPSHOTS E EVOLUÇÃO:
  sheer snapshot --id s1 --repo-path . --ref main
  sheer snapshot --id s2 --repo-path . --ref release-2.1
  sheer evolution --old s1 --new s2

PREFLIGHT IA (PRIVILÉGIO MÍNIMO):
  sheer preflight --role USER_IA_SERVICE --log-dir docs/sheer_audit --snapshot-id s2

BANCO DE DADOS:
  sheer db --verify
  sheer db --list-snapshots
  sheer db --export-csv artifacts/db/audit.csv

GOVERNANÇA:
  # gerado pelo comando `sheer evolution`
  # issue -> docs/issues/0004-evolution-generated.md
  # ADR   -> docs/architecture/adr/ADR-0020-evolution-governance.md
  # blueprint -> docs/BLUEPRINT.md
